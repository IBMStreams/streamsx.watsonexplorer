Operators: Watson Explorer Toolkit

Watson Explorer Toolkit > Operators

Operators

AnalyzeText
This operator uses the Watson Explorer Content Analytics /analysis/text REST API to provide real-time natural language processing (NLP) against ad-hoc text.
PushText
Search
This operator uses the Watson Explorer Content Analytics /search REST API to search for and return documents that satisfy the specified query.
SearchFacet
This operator uses the Watson Explorer Content Analytics /search/facet REST API to search for documents that satisfy the specified query and return only the facets.
SearchPreview
This operator uses the Watson Explorer Content Analytics /search/preview REST API to search for a specific document and return the contents of that document.
WatsonExplorerPush
This operator uses the Watson Explorer Foundational Components REST API to push text to the specified collection.
WatsonExplorerQuery
This operator uses the Watson Explorer Foundational Components REST API to execute queries against a particular collection.
