Operator Search

Watson Explorer Toolkit > com.ibm.streamsx.watsonexplorer 1.0.1 > com.ibm.streamsx.watsonexplorer > Search

This operator uses the Watson Explorer Content Analytics /search REST API to search for and return documents that satisfy the specified query. The query to execute can be specified at compile-time by setting the query parameter, or can be specified dynamically while the application is running by specifying an input attribute via the queryAttr parameter.

More information about the available parameters for this REST API can found in the Content Analytics REST API documentation in the Content Analytics installation directory: <ES_INSTALL_PATH>/docs/api/rest/search.
