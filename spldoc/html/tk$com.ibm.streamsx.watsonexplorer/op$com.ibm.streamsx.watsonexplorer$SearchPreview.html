Operator SearchPreview

Toolkits > com.ibm.streamsx.watsonexplorer 1.1.0 > com.ibm.streamsx.watsonexplorer > SearchPreview

This operator uses the Watson Explorer Content Analytics /search/preview REST API to search for a specific document and return the contents of that document. The document to search for should be specified dynamically at runtime using the uriAttr parameter.

More information about the available parameters for this REST API can found in the Content Analytics REST API documentation in the Content Analytics installation directory: <ES_INSTALL_PATH>/docs/api/rest/search.
